Essential CSS Code

Here are some basic CSS code snippets you should know:

Change Text Color

p {
  color: blue;
}

Change Font Size

h1 {
  font-size: 32px;
}

Set Background Color

body {
  background-color: #f0f0f0;
}

Center Text

div {
  text-align: center;
}

Add Padding

.box {
  padding: 20px;
}

Add Margin

.box {
  margin: 10px;
}
